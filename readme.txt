# Dog Park Recommendations

## Περιγραφή
Ένα WordPress plugin που προτείνει την καλύτερη ώρα για επίσκεψη σε πάρκα σκύλων με βάση τον καιρό και τις συνθήκες του πάρκου. Χρησιμοποιεί προβλέψεις από το Open-Meteo, OpenWeather και Google Weather API.

## Χαρακτηριστικά
- **Gutenberg block** για την εμφάνιση της καλύτερης ώρας επίσκεψης.
- **Κεντρικό μητρώο πάρκων** με προσαρμόσιμες συνθήκες (σκιά, νερό, αποχέτευση, φωτισμός).
- **Καθημερινή ανανέωση** δεδομένων καιρού στις 3:00 π.μ. (EEST).
- **Προσαρμόσιμη βαθμολόγηση** για τη σκυλίτσα σου (π.χ. βάρος στη βροχή, ζέστη, UV).
- **Φόρμα προτάσεων** για επισκέπτες για νέα πάρκα ή βελτιώσεις.
- **Στήριξη για light/dark mode** στον μπλοκ.
- **Branding** με σύνδεσμο στο skilitsa.com.

## Απαιτήσεις
- WordPress 5.8+
- PHP 7.4+

## Εγκατάσταση
1. Κατεβάστε το plugin από το GitHub.
2. Ανεβάστε το στον φάκελο `wp-content/plugins/` του WordPress σας.
3. Ενεργοποιήστε το από τον πίνακα ελέγχου.

## Ρυθμίσεις
Μετά την ενεργοποίηση, μεταβείτε στο `Settings → Dog Park` για να ρυθμίσετε:
- **Βάρη βαθμολόγησης** (βροχή, ζέστη, UV, άνεμος).
- **Ρυθμίσεις παρόχων** (API keys για OpenWeather/Google Weather).
- **Ειδοποιήσεις διαχειριστή** σε περίπτωση αποτυχίας όλων των παρόχων.

## Χρήση
- **Gutenberg Editor**: Πρόσθεσε το μπλοκ **"Dog Park Best Hour"** σε οποιαδήποτε σελίδα.
- **Dropdown**: Επίλεξε ένα πάρκο από τη λίστα.
- **Front-End**: Εμφανίζει την καλύτερη ώρα, εύρος θερμοκρασίας και dropdown με άλλες καλές ώρες.

## Προτάσεις Επισκεπτών
Χρησιμοποιήστε το μπλοκ **"Dog Park Suggestion Form"** για να επιτρέψετε στους επισκέπτες να προτείνουν νέα πάρκα ή βελτιώσεις. Οι προτάσεις αποθηκεύονται στον πίνακα ελέγχου και αναθεωρούνται από τον διαχειριστή.
